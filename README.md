# App movil - App Repuestos EC
Angular - Ionic <br>
Realización de una App Móvil del último proyecto de la materia de Desarrollo de Aplicaciones Web y Móviles Paralelo 1
## Install the Ionic app

```bash
npm install
```
## Run the Ionic app on the browser

```bash
ionic serve
```
